Gazetteer of the World

Feature type
About this demonstrator

A Gazetteer of the World (Royal Geographical Society, 1856; 7 volumes, public domain) turned into structured, typed, geolocated linked data — and explored here with no application server (a static GitHub Pages site: vector-tile map, whole-corpus reader, in-browser search).

Source & OCR

Built from the public-domain HathiTrust page scans. No inherited OCR (licence + quality): the pages were re-OCR'd with the layout-aware Surya model on the Pitt CRC GPU cluster, with two-column reading order reconstructed from line geometry.

How it was made

Every record is typed by a self-hosted Llama-3.3-70B (closed Getty-AAT enum); tables and plates are digitised by Qwen2.5-VL; each place is reconciled to the World Historical Gazetteer by a containment-aware cascade (resolve the admin parents to polygons, then match inside the parent region). All LLM/VLM stages are self-hosted — no per-token cost. 116,292 places · 84.8% reconciled · 1,774 tables · 133 plates.

Caveats — please read

This is a research demonstrator, not an authority. Reconciliation to the WHG is fully automatic and a work in progress: a great many matches are wrong (a same-named place elsewhere, or no real equivalent at all), and automatic matching of historical names will never be fully resolved. Table and plate detection is patchy — some are missed, mislocated, or mis-transcribed. Treat every located point, match, and digitised table as a suggestion to verify against the source page (each record links to its scan), not as ground truth — and please flag errors with the ⚑ Report link in any popup or reader entry.

Origin & credits

The project arose from Humphrey Southall's offer of his transcripts to Ruth Mostern and the resulting discussion; it uses its own public-domain OCR rather than those transcripts. Source code & method: github.com/WorldHistoricalGazetteer/gazetteer-of-the-world.
