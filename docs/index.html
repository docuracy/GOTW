A Gazetteer of the World → Linked Data

Transforming a 19th-century gazetteer into a structured authority gazetteer for the World Historical Gazetteer.

This project parses the OCR'd transcripts of A Gazetteer of the World (c. 1856) into structured, AAT-typed place records, geolocated against the World Historical Gazetteer via its Reconciliation API.

Coming here

An interactive MapLibre demo plotting a sample of extracted places, geolocated through the WHG Reconciliation API.

Pipeline status

  • ✓ HTML parsed to a structured store (entries, cross-refs, tables, page provenance)
  • ✓ Toponyms normalised from UPPERCASE to title case
  • ✓ Getty AAT feature-type shortlist drafted and validated
  • … LLM extraction of places + disambiguation context
  • … WHG reconciliation, table recovery, and map extraction from the page scans

Worked example — cost vs quality of extraction models (Claude Haiku vs Gemini Flash / Flash-Lite): an evidence base for choosing an LLM for a historical-PDF → linked-data pipeline.

Source code and method: github.com/docuracy/GOTW.

Built on OCR transcription overseen by Prof. Humphrey Southall (University of Portsmouth; GB Historical GIS). Reconciliation indices courtesy of the World Historical Gazetteer (University of Pittsburgh).
